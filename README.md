Customer Churn Analysis & Risk Segmentation

Project Overview
This project analyses customer churn behavior using Excel. The goal was to identify key drivers of churn and segment customers based on risk level.

Key Insights
21% of active customers are high-risk
Customers inactive for 2–3 months are more likely to churn
Low transaction activity strongly correlates with churn

Tools Used
Microsoft Excel (PivotTables, Dashboard, Data Analysis)

Business Recommendations
Implement early warning systems
Target high-risk customers
Introduce loyalty incentives
